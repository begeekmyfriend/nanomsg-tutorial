nanomsg tutorial
================

There are examples of IPC message passing methods supported by nanomsg.

Build nanomsg library
---------------------

See [nanomsg](https://github.com/nanomsg/nanomsg)

Build examples
--------------

```shell
        cd nanomsg-examples
        make
```

Pipeline
--------

```shell
        ./pipeline.sh
```

Request & Reply
---------------

```shell
        ./reqrep.sh
```

Pair
----

```shell
        ./pair.sh
```

Publisher & Subscriber
----------------------

```shell
        ./pubsub.sh
```

Survey
------

```shell
        ./survey.sh
```

Bus
---

```shell
        ./bus.sh
```
